This plugin allows you to insert an image gallery retrieving images from Flickr.

== Description ==
FlickrPhotogallery allows you to insert one (or more) image gallery (retrieved from Flickr) into your posts using a shortcode;
You can also use the built-in widget to insert the gallery into the sidebars.

The plugin allows to retrieve:
* Public images
* User-specific images
* Photoset-specific images
* Group images

It is also possible to choose to show the image gallery as an unordered list or as a slideshow, in this case you can select the dimension of it.

== Installation ==
The installation FlickrPhotogallery is quite simple:
1. Unzip flickrphotogallery.zip
2. Upload the flickrphotogallery folder into the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in Wordpress
4. Enjoy
